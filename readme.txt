How to compile SASS in miliseconds using sassc-bin on Windows?


1. Install Node.js from https://nodejs.org/en/download/


2. Install nodemon to monitor SASS file directory for changes

npm install -g nodemon


3. Install bless v3 to split a compiled css file into chunks for older IE compatibility

npm install -g nodemon@3.0.3


4. Run watch.bat to monitor and recompile
